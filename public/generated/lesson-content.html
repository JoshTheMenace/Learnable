Introduction to Bubble Sort

Bubble Sort is one of the simplest sorting algorithms and a perfect starting point for anyone new to algorithmic thinking. It works by repeatedly stepping through a list, comparing adjacent items and swapping them if they are in the wrong order. Think of it like the way bubbles rise to the surface of water – the largest values “float” to the top of the list after each pass.

How Bubble Sort Works

The algorithm follows a very straightforward process:

  • Compare each pair of adjacent elements.
  • Swap them if they are out of order.
  • After each full pass, the largest unsorted element is placed in its final position.
  • Repeat the passes until no swaps are needed.

A Simple Python Example 

def bubble_sort(arr):
    n = len(arr)
    # Traverse through all array elements
    for i in range(n):
        # Last i elements are already sorted
        for j in range(0, n-i-1):
            # Swap if the element found is greater
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

# Example usage
numbers = [64, 34, 25, 12, 22, 11, 90]
print(bubble_sort(numbers))

Run the code above and watch how the list gradually becomes sorted after each pass. Even though Bubble Sort isn’t the fastest for large datasets, its clarity makes it an excellent teaching tool.
